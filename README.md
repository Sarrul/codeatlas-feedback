# CodeAtlas Feedback & Issues

Welcome! 👋  

This repository is used to track:
- 🐛 Bug reports  
- 💡 Feature requests  
- 🚀 Improvement suggestions  

If you're experiencing issues with CodeAtlas or have ideas to improve it, please open a new issue using the appropriate template.

## Before Submitting

Please:
- Make sure the issue hasn’t already been reported
- Include clear reproduction steps (for bugs)
- Attach screenshots if possible
